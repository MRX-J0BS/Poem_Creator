## 项目简介

欢迎来到这个项目，本项目的初衷是通过枚举来实现批量的诗词生成（尽管大多都没有意义

##### 目前的项目核心

另外本项目目前使用的随机模块是random，通过字符列表中的随机抽取生成诗词，如果需要复原则再次调用random的种子（众所周知random是伪随机）

**当前main function包含内容：**random任意格式诗词生成、random任意格式诗词复原
